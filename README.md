# DZ-Steg-Challenges
A collection of short steganography challenges compiled for use in the DayZero Cyber Club.


## Instructions
Challenge files will cover a series of different steg techniques, from simply hiding flags in image data, to LSB embedding, to audio steg. All the necessary files for this challenge have been provided through this repo. 
It is up to the challenger to determine which tools they will need for image steganography. However, the audio challenges were hidden using a particular [niche tool](https://github.com/Jpinsoft/DeepSound).

All flags will follow the format DZCTF{Flag_String_Here}.
